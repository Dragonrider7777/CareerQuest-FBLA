Career Hub Description with Logic

The main navigation screen of the game. It acts as a central map where players choose which career world to explore.

**LAYOUT**
Simple, cartoon-style layout with:
- A neutral background (Possibly sky blue?)
- Large, colorful rectangular panels that each represent a different career
	Each rectangle includes:
	- Career name (large font size and bolded)
	- A short caption of business skill learned in the minigame
	- Possibly an icon??

**LOGIC**
- Each rectangle is a clickable button
- The game opens a Career Info Scene when a rectangle is clicked
	ALWAYS ALLOW PLAYERS TO RETURN TO THE HUB

The Career Info Scene serves as an informational screen about the selected career
Includes:
- Career name at top
- A short description what the job is like
- Key skills required for the career (foreshadowing for what the minigame is about)
- 2 large buttons:
	- Play Career
	- Back to Hub
